• accesskey = Untuk memberi shortcut keyboard ke elemen. class = Untuk memberi nama kelas yang bisa digunakan di CSS atau JavaScript. id = Untuk memberi identitas unik ke elemen (hanya boleh satu dalam satu halaman). dir = Untuk menentukan arah teks: kiri ke kanan atau kanan ke kiri. lang = Menentukan bahasa konten dalam elemen. style = Untuk menulis CSS langsung di elemen (inline style). tabindex = Menentukan urutan elemen saat pencet tombol Tab. title = Menampilkan info tambahan saat kursor diarahkan ke elemen (tooltip).
